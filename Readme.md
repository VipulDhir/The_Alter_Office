# Analytics API

This is a simple analytics platform for collecting events, generating API keys, and viewing analytics summaries and user stats.

---

## Getting Started

### Prerequisites
- Node.js and npm installed
- Docker Desktop installed

### Running Locally
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-folder>
   npm install

   Command to Run this : docker-compose up --build
   Note : Run Docker Desktop in Background
