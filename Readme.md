# Analytics API

This is a simple analytics platform for collecting events, generating API keys, and viewing analytics summaries and user stats.

---

## Getting Started

### Prerequisites
- Node.js and npm installed
- Docker Desktop and Docker Compose installed

### Running Locally
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-folder>

2. Install Dependencies 
   ```bash
   npm install

3. Enter Your MySQL Password in src/config file, docker-compose , .env
   ```bash
   Command to Run this Application : docker-compose up --build

   ```bash
   Note : Run Docker Desktop in Background
